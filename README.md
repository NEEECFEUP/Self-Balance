Motivação:
  Projeto realizado com o intuito de colocar em prática alguns conhecimentos sobre Controlo.

Descrição:
  Este projeto consiste em criar um sistema autónomo capaz de equilibrar uma bola no centro de uma calha, ao mesmo tempo que tenta prever o seu estado.

Entrada:
  A posição inicial da bola pode ser qualquer parte da calha.
  O sistema recebe como entrada a posição da bola na calha, recorrendo a um sensor ToF.

Saída:
  O sistema atua apenas sobre o ângulo que a calha faz com a horizontal, através de um servo motor no centro da mesma.
  O sistema utiliza uma tira de LEDs RGB para comparar em tempo real a posição real da bola vs. a posição que ele prevê.
